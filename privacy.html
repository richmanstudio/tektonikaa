Политика конфиденциальности

Дата вступления в силу: 31 мая 2025 года

1. Общие положения

Настоящая Политика конфиденциальности (далее — Политика) определяет порядок обработки персональных данных пользователей веб-сайта tectonica.ru (далее — Сайт) ООО «ТЕКТОНИКА» (далее — Оператор).

Используя Сайт, Пользователь дает согласие на обработку своих персональных данных в соответствии с настоящей Политикой.

2. Собираемые данные

Оператор может собирать следующие категории данных:

  • Данные, предоставляемые Пользователем: имя, контактные данные (email, телефон), содержимое сообщений при заполнении форм на Сайте
  • Технические данные: IP-адрес, тип браузера, операционная система, данные о посещенных страницах, время посещения
  • Данные cookie: файлы, сохраняемые браузером для улучшения работы Сайта

3. Цели обработки данных

Оператор использует персональные данные для следующих целей:

  • Обработка запросов и обращений Пользователей
  • Предоставление информации об услугах компании
  • Улучшение работы Сайта и пользовательского опыта
  • Анализ статистики посещаемости Сайта
  • Выполнение обязательств перед Пользователями

4. Передача данных третьим лицам

Оператор не передает персональные данные Пользователей третьим лицам, за исключением следующих случаев:

  • Получено явное согласие Пользователя
  • Требование законодательства РФ
  • Для обеспечения функционирования Сайта (хостинг-провайдеры, аналитические сервисы)

5. Защита данных

Оператор принимает необходимые организационные и технические меры для защиты персональных данных от неправомерного доступа, изменения, раскрытия или уничтожения.

6. Cookies

Сайт использует файлы cookie для:

  • Запоминания предпочтений Пользователя
  • Сбора аналитических данных
  • Обеспечения безопасности

Пользователь может отключить использование cookie в настройках своего браузера.

7. Права Пользователя

Пользователь имеет право:

  • Получать информацию об обработке своих персональных данных
  • Требовать уточнения, блокирования или уничтожения данных
  • Отозвать согласие на обработку данных
  • Обратиться в уполномоченный орган по защите прав субъектов персональных данных

8. Изменения в Политике

Оператор оставляет за собой право изменять настоящую Политику. Актуальная версия Политики всегда доступна на Сайте.
